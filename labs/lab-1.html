Purpose of This Lab

This first lab is designed to help you review data wrangling and visualization in Python in preparation for machine learning workflows you will use throughout the course. You will review key skills from DATA 271, practice reading and navigating a computational notebook, and begin building a personal quick reference you can return to all semester.

This lab is not about learning new machine learning algorithms. Instead, the focus is on:

  • Translating information into data a machine can use

  • Understanding how data is structured for the libraries we’ll be using

  • Reviewing essential wrangling and visualization skills
